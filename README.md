# Garmin Data Extractor

The Garmin Connect website shows various health-related data, but not all of it is exportable.

This browser extension aims to allow you to download data from some of the metrics that are not currently supported by the Garmin Connect Website.

Supported metrics:

- Heart Rate: get the daily records shown on the screen

Output:

The metrics are downloaded as a `csv` file, that can be imported into a spreadsheet software or analysed in a different way.
